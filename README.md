# AI-Learning-Roadmap-Generator

AI Learning Roadmap Generator is a simple web application that creates a personalized AI learning roadmap for users based on their name, current skill level, and goals.
The roadmap is generated using Large Language Models (LLMs) through the Groq API, and the interface is built with Gradio.

This project helps beginners and intermediate learners understand what to learn step-by-step in Artificial Intelligence and Machine Learning.

Generate personalized AI learning roadmap
User-friendly Gradio web interface
